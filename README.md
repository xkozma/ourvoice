# OurVoice

A demo civic engagement web app where citizens can compare government law voting outcomes with citizen sentiment.

Quick start:

```bash
npm install
npm run install:all
npm run dev
```

See [RUNNING.md](./RUNNING.md) for setup and local run instructions.

## Render (single service)

- Build command: `npm install && npm run render:build`
- Start command: `npm run dev`
- Required env: `JWT_SECRET`
